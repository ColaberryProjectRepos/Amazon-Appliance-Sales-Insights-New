Creating Waterfall Chart 22

  •  Create a measure description table.

  • On the Power BI Desktop Home ribbon, Click on Enter data, name the first column  SubKPI's , the second column as Index and table name as SubKPI's.
      Insert the following data in the table. SubKPI's(Sales YoY%, Rev YoY%, OoS YoY%,              Rating YoY% ) and Index(1,2,3,4) consecutively as shown above.

  •  Follow the steps in the GIF provided below.


  •   Create Measure 
WaterFallValue = SWITCH(SELECTEDVALUE('SubKPI''s'[Index]),1,[Sales YoY%],2,[Rev                                        YoY%],3,[OoS YoY%],4,[Rating YoY%])

  • Copy and Paste year chart.
  • From the visualizations pane select Waterfall Chart
  • Fill up the figures provided below for the necessary fields.
  • Category= SubKPI's from subkpi's table (Right click on it->select rename for this visual->write "Measure"
  • Y-axis= WaterFallValue from All measures table(Right click on it->select rename for this visual-> write "Value"
