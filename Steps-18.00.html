Adding Date Slicer 18



1. Click on an empty space on the canvas, and from the visualizations pane select Slider.
2.  Under the Fields Pane from Calendar Table drag year column to the fields value box
3.  Fill up figures provided below for the necessary fields
Visual: 
       Slicer settings:
            Style=Between, Slicer header=off, 
             values-> Font=DIN Size=11, color =#000000, Slider= ON
General:
       properties: 
            Size: Height=109, Width=212    
            Position: Horizontal=0, Vertical =76 
            Title=On, Font DIN, Size=14, Text color=#FFFFFF, Background color=#FF9900,
             Horizontal alignment=center 
             Effects: Background=On, Color =#000000, Transparency=100%, Visual border                 =On, Color=#FFC266.
