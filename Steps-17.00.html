Adding Title 


1. Go to the "Home" tab in the Power BI ribbon and click on the "Text box" icon. This will allow you to insert a text box onto your report canvas.
2. With the text box selected, you'll see the "Format" tab in the "Visualizations" pane on the right. Then click on General-> 
  properties:
        Size: Height=70, Width=648
         position: Horizontal=212, vertical =1  

2. Click inside the text box and type in " Amazon Appliance Sales Insights."
   Adjust (Font=DIN, Font Size=32 and Color =#000000)

Adding Refreshed Date


1. Click on the report and rom the visualizations pane select Visual Card.
2.  Drag the Refreshed Date from Refresh Date Table to the fields value box.
3.  In the Formatting Tab-->Visual-->Callout value:
     Font-DIN, Size -13, Color =#000000
     Category label=Off
General:
     Size: Height=70, Width=219
     Position: Horizontal=70, Vertical=1
     Title: ON, Type "Refreshed Date", Font =Segoe UI Bold, Size = 14, Text Color =#000000, Background color=#FFFFFF, Horizontal alignment = center
