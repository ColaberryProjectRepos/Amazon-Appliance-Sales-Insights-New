Creating 'All Measures' table 14

Creating 'All Measures' table which has (main KPIs, Sub KPIs ,waterFallValue and Arrows)



1.  Go to “Report view” in Power BI.
2.  In Home Tab, select Enter data and Create New table
3. Rename the table to ‘All Measures’ Table
4. Click Load
5. Right Click on 'All Measures' and select New Measure
5. Copy and paste the DAX formula for 'TotalSales' Measure on the Dax Bar 
6. Click on the Right mark.
7. Repeat step 5-6 until you create all measures using the below Dax formula.
Follow the following Gif for step by step instruction.

Total Sales = SUM ('Amazon Appliance Sales'[monthlyUnitsSold])

Measures:
1. Total Sales = SUM ('Amazon Appliance Sales'[monthlyUnitsSold])
 
     Format-Whole Number with 0 places and use Comma (,)
2. Sales per Mth = DIVIDE([Total Sales],DISTINCTCOUNT('Calendar'[Month]),0)
    Format-Whole Number with 0 places and use comma(,)
3. Sales YoY% = VAR CurrentSales = [Total Sales]
VAR PreviousSales = CALCULATE( [Total Sales],
    SAMEPERIODLASTYEAR('Calendar'[Date]))
RETURNDIVIDE(  CurrentSales - PreviousSales,    PreviousSales,  0)
    Format -percentage with 1 places.
4. Total Revenue = SUM('Amazon Appliance Sales'[MonthlyRevenueEstimate])
    Format -Currency with 0 places and use comma(,)
5. Rev per Mth = DIVIDE([Total Revenue],DISTINCTCOUNT('Calendar'[Month]),0)
  Format -Currency with 0 places and use comma(,)
6. Rev YoY% = VAR CurrentYearRevenue = [Total Revenue]
VAR PreviousYearRevenue =    CALCULATE ( [Total Revenue],        SAMEPERIODLASTYEAR ( 'Calendar'[Date] )    )
RETURN
    DIVIDE ( CurrentYearRevenue - PreviousYearRevenue, PreviousYearRevenue, 0 )
  Format -percentage with 1 places.
7. Out of Stock% = DIVIDE(
    COUNTROWS(FILTER('Amazon Appliance Sales', 'Amazon Appliance Sales'[OutOfStockNow] = TRUE)),
    COUNTROWS('Amazon Appliance Sales'),    0)
Format -percentage with 1 place.
8. OoS YoY% = 
VAR CurrentYearOoS = [Out of Stock%]
VAR PreviousYearOoS =
    CALCULATE (  [Out of Stock%],
        SAMEPERIODLASTYEAR ( 'Calendar'[Date])    )
RETURN
    DIVIDE ( CurrentYearOoS - PreviousYearOoS, PreviousYearOoS, 0 )
 Format -percentage with 1 place.
9. Avg Rating = AVERAGE('Amazon Appliance Sales'[ReviewRating])
  Format-Decimal with 1 place.
10. Rating YoY% = 
VAR CurrentRating = [Avg Rating]
VAR PreviousRating = CALCULATE(
    [Avg Rating],    SAMEPERIODLASTYEAR('Calendar'[Date]))
RETURN
DIVIDE(CurrentRating - PreviousRating, PreviousRating, 0)
Format -percentage with 1 places.
11. Arrow Sales Yoy% = SWITCH(TRUE(),[Sales YOY%]<0,UNICHAR(9660),UNICHAR(9650))
12. Arrow Rev Yoy% = SWITCH(TRUE(),[Rev YoY%]<0,UNICHAR(9660),UNICHAR(9650))
13. Arrow Rating Yoy% = SWITCH(TRUE(),[Rating YOY%]<0,
          UNICHAR(9660),UNICHAR(9650))
14. Arrow Oos Yoy% = SWITCH(TRUE(),[OoS YoY%]<0,
         UNICHAR(9660), UNICHAR(9650))
15. WaterFallValue = SWITCH(SELECTEDVALUE('SubKPI''s'[Index ]),1,[Sales YoY%],2,[Rev YoY%],3,[OoS YoY%],4,[Rating YoY%])
