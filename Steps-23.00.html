Subcategory chart using Line and Stacked column chart



  • Copy and Paste year chart.
  • From the visualizations pane select Line and Stacked column chart
  • Fill up the figures provided below for the necessary fields.
  • Category= SubcategoryName from Amazon Appliance Sales table (Right click on it->select rename for this visual->write "Subcategory"
  • Column y-axis=Sales per Mth 
  • Line y-axis=Sales YoY%, Rev YoY%, OoS YoY%, Rating YoY%
  •  Tooltips= Sales YoY%, Rev YoY%, OoS YoY%, Rating YoY%, Total Sales,                                         Total Revenue, Rev per Mth, Out of Stock%, Avg Rating.

  • Visual:      
                X-axis=on, Values: Font=DIN size=11, color=#000000, Title =off     
             Y-axis=on, Values: Font=DIN size=11, color=#000000, Title=off             
             Secondary y-axis=on,  Font=DIN size=11, color=#000000, Title=off       
             Legend=on, Position =Top left, style=marker (circle default), Match line                                       color=on, Font=DIN, size=11, color=#000000
             Gridlines=Horizontal on, Style= Solid, color =#FFFFFF, vertical=off, Zoom                                     slider=off
              Columns: Color =#FF9900
               Lines: Series=All, Shape=solid, Join type=Round, Colors(Rating YoY%=                                     #F083BA, OoS YoY%=#205FA2, Rev YoY%=#67002E,                                                             SalesYoY%=#9023DE , YoY%=#9023DE                                                                      Markers=off, Date labels=off, Series labels=off   
  • General:     
                Properties: Size (Horizontal=321, Vertical=355), Position                                                      (Horizontal=357, Vertical=398), Title=on (Text="Subcategory ", Font=DIN,                                        Size=14, color=#FFFFFF, background color= #FF9900, Horizontal                                      alignment=center)   Effects: Background=On, Color =#000000,                                           Transparency=100%, Visual border =On, Color=#FFC266,                                                   Tooltips=on
     Go to filters ->Filters on this visual select for Subcategory(Filter type =Top N, Show items =Top12, By value= Rev per MTh) then click on Apply filter.



Creating Brand chart using Scatter plot



  • Click on an empty space on the canvas, and from the visualizations pane select Scatter chart 
  • Drag SubcategoryName column from Amazon Appliance Sales table to Values (Right click on it-select rename for this visual->write "Subcategory")
  • Drag BrandName column from Amazon Appliance Sales table to values. (Right click on it-select rename for this visual->write "Brand")
  • Fill up the figures provided below for the necessary fields.
  •  Column X-axis=Total Sales
                    Y-axis= Avg Rating.
                    Size= Total Revenue
                     Tooltips= Sales YoY%, Rev YoY%, OoS YoY%, Rating YoY%, Total Sales, Total                        Revenue, Rev per Mth, Out of Stock%, Avg Rating.
  • Visual:   
             X-axis=on,  Values: Font=DIN size=11, color=#000000, Title                                             =on, color=#118DFF
             Y-axis=on, Values: Font=DIN size=11, color=#000000, Title=on  ,color                                            =#CCCF18      
             Legend=on, Position =Top left, Font=DIN, size=11, color=#000000, title="Out                                          Of Stock%"
             Gridlines=Horizontal on, Style= Solid, color =#FFFFFF, vertical=off, Zoom                                     slider=off
              Columns: Color =#FF9900
                Markers=on(Format Style=Gradient, What field=Out Of Stock%, Minimum                                       color=#D2F3F1, center=0ACEC1, Maximum=#023F3B)
                  Category Label=on(Font=DIN, size=9,color=#000000)
  • General:     
                 Properties: Size (Horizontal=321, Vertical=355), Position                                                     (Horizontal=924, Vertical=395), Title=on (Text="Brand ", Font=DIN,                                    Size=14, color=#FFFFFF, background color= #FF9900, Horizontal                                     alignment=center)   Effects: Background=On, Color =#000000,                                          Transparency=100%, Visual border =On, Color=#FFC266, Tooltips=on
